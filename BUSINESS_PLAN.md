## 💰 ビジネスモデル比較

### モデル1: テンプレート販売（買い切り）⭐ 推奨スタート

**概要:** ソースコードを販売、購入者が自分でデプロイ

| 項目 | 詳細 |
|------|------|
| **価格** | ¥4,980〜¥9,800（買い切り） |
| **販売場所** | Gumroad / LemonSqueezy / BOOTH |
| **必要な準備** | README整備、セットアップガイド |
| **開発コスト** | 低（現状のまま） |
| **運用コスト** | ほぼゼロ |
| **収益性** | 中（販売数次第） |

**含めるもの:**
- ✅ 完全なソースコード
- ✅ 詳細なセットアップガイド（日本語）
- ✅ Vercelデプロイガイド
- ✅ API設定ガイド（Groq, Gemini, Google）
- ✅ 商用利用ライセンス
- ✅ 6ヶ月間のメールサポート

**メリット:**
- 👍 すぐ始められる
- 👍 サーバー費用なし
- 👍 顧客サポート少ない

**デメリット:**
- 👎 一回限りの収入
- 👎 アップデート提供が難しい

---

### モデル2: SaaS（月額課金）⭐ 成長フェーズ向け

**概要:** ユーザーがログインして使うサービス（Notionのような形態）

| 項目 | 詳細 |
|------|------|
| **価格** | 無料/¥500月/¥1,200月 |
| **必要な開発** | 認証、データベース、課金システム |
| **開発コスト** | 高（2-4週間追加開発） |
| **運用コスト** | 中（サーバー、DB） |
| **収益性** | 高（継続課金） |

**必要な追加開発:**
- ユーザー認証（NextAuth → 個別アカウント）
- データベース（Supabase / PlanetScale）
- 課金システム（Stripe）
- ダッシュボード管理画面

**プラン例:**
| プラン | 価格 | 機能 |
|--------|------|------|
| **Free** | ¥0 | 基本機能、広告あり |
| **Pro** | ¥500/月 | 全機能、AI 100回/日 |
| **Premium** | ¥1,200/月 | 無制限AI、優先サポート |

**メリット:**
- 👍 継続的な収入
- 👍 ユーザーデータ管理可能
- 👍 アップデート提供しやすい

**デメリット:**
- 👎 開発に時間がかかる
- 👎 サーバー費用が発生

---

### モデル3: フリーミアム + アフィリエイト

**概要:** 無料で使えるが、APIキー取得時にアフィリエイト収入

| 項目 | 詳細 |
|------|------|
| **価格** | 無料 |
| **収益源** | アフィリエイト、スポンサー |
| **開発コスト** | 低 |
| **収益性** | 低〜中 |

**収益源:**
- Vercel Proへの紹介リンク
- AI API有料プランへの誘導
- 英語学習サービスとの提携
